# Credit Card Fraud Detection

This project detects fraudulent credit card transactions using Logistic Regression in Python.

## Dataset
Source: Kaggle - Credit Card Fraud Detection
284,807 transactions with only 492 fraud cases (highly imbalanced).

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## How it Works
1. Load and explore the dataset
2. Balance the data using under-sampling
3. Scale features using StandardScaler
4. Train a Logistic Regression model
5. Evaluate using accuracy, precision, recall and confusion matrix

## How to Run
1. Download the dataset from Kaggle
2. Upload the notebook to Google Colab
3. Upload the dataset file when prompted
4. Run all cells in order

## Future Improvements
